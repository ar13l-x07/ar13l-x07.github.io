moo

moo is a collection of easy to integrate C++ tools and treats. Elements of moo are contained within a single .h header file and the only depend on the C++ STL.

  • moo::format

Tools

Move-TODOBlocks

PowerShell script that moves TODO comments from source code to a TODO.shelf (and vice-versa). Available on Windows.

Webp-Convertor

WebP conversion script, converts .webp images to .png images. Available on Windows.

FibonacciHeap

A C++ single header Fibonacci Heap, this is an optimized priority queue, very useful for path-finding algorithms.
